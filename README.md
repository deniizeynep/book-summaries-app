# Book Summaries App

## Description

Book Summaries App was made using [React Native](https://reactnative.dev/) and CSS. 

This app lists the best-selling books worldwide with their covers and summaries. Books can be deleted and new ones added.

## Getting Started

###  Clone the repository

```bash
git clone https://github.com/deniizeynep/book-summaries-app.git
```

###  Install dependencies

```bash
npm install
```

###  Run the project locally

```bash
npm run start
```

###  Run the project on Android

```bash
npm run android
```
